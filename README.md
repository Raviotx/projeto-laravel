# projeto-laravel

Projeto de **Lista de Tarefas (To-do List)** desenvolvido em **Laravel**, com foco em aprendizado de rotas, controllers, views, migrations e banco de dados SQLite.

O sistema permite:
- Criar tarefas
- Listar tarefas
- Editar tarefas
- Excluir tarefas

Projeto acadêmico.

---

## Tecnologias usadas

- PHP 8+
- Laravel
- SQLite
- Blade
- Composer
